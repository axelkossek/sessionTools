Tool used to ease generation of NPCs during session.
Data format .json.

Functionalities:
- Generate random NPC
- Create file containing NPCs generated during specific Session
	- Name
	- Surname / Clan
	- Stats
	- Occupation
	- Appearance
- Append geneated NPC into beforementioned file
- List NPCs in human readable format
- Remove appended NPCs
- List of names, stats and races
- Update generator conditions based on region file
	- Racial distribution
	- Status distribution
	- Gender distribution
